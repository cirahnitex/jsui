We are going to develop a software to highlight the user’s interests when browsing the internet.

We decided to build this software using HTML and Javascript. The reason is quite simple. Since the project is “for the web”, we would like to make it “by the web”.

Stage 1: Basic Skeleton

user can browse the web via our software

our software can record user’s action.(mouseover, click)

our software can manipulate this web page

Stage 2: Obtaining Hints

define a hotkey for user to “add to interests” and “remove from interests”

identify the exact element that the user want to “add” or “remove”

store all the sample elements that user submits

Stage 3: Creating Selectors

for each web page, analyze the positive sample elements and negative sample elements that user submits, figure out a Selector to locate the user’s interests in this web page..

store these Selectors for later use

Stage 4: Grouping up web pages

for web pages that share similar or identical selectors, find their common Features.

store these Features for later use

Stage 5: Applying all that we’ve got

create a “smart browsing mode”

when user is browsing the web in smart browsing mode, check the Features for current web page, and apply corresponding Selectors. Highlight the elements that our Selector selects.
